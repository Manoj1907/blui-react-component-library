# React Themes Overview

This package, `@brightlayer-ui/react-themes`, provides theming support for Eaton applications using the Brightlayer UI design system. It includes resources for developers using React with [MUI's Material UI](https://www.npmjs.com/package/@mui/material) version 5+. This package comes with two theme options: a Blue theme (standard) and a Dark theme.

# Installation

We encourage you to use [Brightlayer UI CLI](https://brightlayer-ui.github.io/development/cli) to generate a new project. The CLI is a tool that can generate a blank React project that includes all essential Brightlayer UI packages, including the Brightlayer UI themes.

To integrate the package into an existing project, you may install it with npm:

<CodeBlock code={`npm install --save @brightlayer-ui/react-themes`} language={'sh'} />

or yarn:

<CodeBlock code={`yarn add @brightlayer-ui/react-themes`} language={'sh'} />
